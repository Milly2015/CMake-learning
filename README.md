CMake learning
==============

Quick Start
-----------
We have two files named "test.cpp" and "CMakeLists.txt" under the current
directory. To build the project `hello`, run the folling commands:
```bash
$ cmake -H. -Bbuild
$ cmake --build build -- -j3
```
